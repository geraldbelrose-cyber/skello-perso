
# Mini‑Skello (prototype) — Application locale (PC)

Fonctions :
- Planning (génération semaine + modification manuelle)
- Absences / congés
- Retards
- Rapports d'heures (export CSV)
- Paramètres modifiables : horaires + 1 jour de repos / semaine + 1 samedi off / mois

## Installation
1) Installer Python 3.10+
2) Dans un terminal, dans le dossier du projet :

```bash
pip install -r requirements.txt
```

3) Lancer :

```bash
streamlit run app.py
```

## Données
- Stockées en local dans `data.db`
